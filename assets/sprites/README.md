# Sprites Folder

Place your sprite assets here.

## Required Sprites

From CraftPix.net freebies (https://craftpix.net/freebies/), look for:

1. **Player Character**
   - Idle, run, jump, shoot animations
   - File: `player.png` or `player_sheet.png` (spritesheet format preferred)
   - Size: ~32x48 pixels per frame recommended

2. **Enemy Characters**
   - Walking patrol animations
   - File: `enemy.png` or `enemy_sheet.png`
   - Size: ~32x40 pixels per frame

3. **Bullet/Projectiles**
   - Simple bullet sprite
   - File: `bullet.png`
   - Size: ~8x4 pixels

4. **Menu Background** (optional)
   - Background image for start/pause menus
   - File: `menu_bg.png`
   - Will be auto-scaled to screen size

## Recommended Free Packs on CraftPix

- "Free Platformer Game Tileset Pixel Art" - may include character sprites
- Search for "Free Pixel Art Sprites" or "Free Platformer Characters"
- Look for packs with walk/run/jump animations

## Format Notes

- PNG format preferred (supports transparency)
- Spritesheets (multiple frames in one image) work well
- Individual frame images also supported
